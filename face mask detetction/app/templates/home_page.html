About

Face Mask Detector


This project is made with an aim to enforce the mandate for wearing masks in public places following Covid19 Norm and to make a working model for accurate mask detection by image processing. With this web app we want to track percentage of people not wearing face masks and area where less mask is detected to restrictions in specific areas only

For now we can see the working model of our project where user can turn on their system's web camera and can detect number of faces wearing masks and no of faces not wearing masks.
